# Mida tehti

Kirjelda konkreetselt tehtud tegevused.

Väldi üldsõnalisi lauseid:

- "seadistati süsteem"
- "tehti vajalikud muudatused"

Too välja:

- milline tegevus tehti
- millise käsu või tööriistaga

Näide (halb):

- seadistati server

Näide (hea):

- paigaldati tarkvara käsuga `sudo apt install pakett`
